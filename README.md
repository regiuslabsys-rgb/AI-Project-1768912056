"# Менеджер Чат-ботов\n\n## Описание\n\nВеб-сервис для управления чат-ботами. Этот MVP предоставляет базовый интерфейс для создания, редактирования и развертывания чат-ботов.\n\n## Функциональность\n\n*   Главная страница:\n    *   Список доступных чат-ботов (фиктивный).\n    *   Кнопка \"Создать нового чат-бота\".\n*   При нажатии на кнопку \"Создать нового чат-бота\" отображается сообщение \"Функция в разработке\".\n\n## Технологии\n\n*   HTML\n*   CSS (Tailwind CSS)\n*   JavaScript (ванильный)\n\n## Запуск\n\n1.  Откройте `index.html` в вашем браузере.\n\n## Планы на будущее\n\n*   Реализация страницы создания/редактирования чат-бота.\n*   Функциональность сохранения и редактирования чат-ботов.\n"
